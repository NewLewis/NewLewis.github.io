中位数&第k小的数

作者:PJR    发布于: 2018-06-19

今天终于有时间来记一下BIGO面试遇到的一个问题，当时给了个O(2n)的算法，面试官也没跟我讨论一下，我还以为只是一道考查代码实现能力的一道题，现在想起来真的蠢到家了。

中位数

描述

给定两个等长的有序序列，找这两个序列的上中位数A[L/2]。

当初的我

当时的做法：归并，然后直接找到A[L/2]。时间复杂度O(2n),额外空间复杂度O(2n)。时间复杂度主要是归并的操作，空间复杂度也是因为归并的时候需要他们一共长的空间。

当时给的代码：

void merge(int a[],int b[],int n,int c[]){
    int i = 0,j = 0,k = 0;
    while(i < n && j < n){

        while(a[i] <= b[j])
            c[k++] = a[i++];

        while(a[i] > b[j])
            c[k++] = b[j++]; 
    }

    while(i<n)
        c[k++] = a[i++];

    while(j < n)
        c[k++] = b[j++];
}

int solve(int a[],int b[],int n){
    int c[n*2+1];
    merge(a,b,n,c);
    return c[n-1];
}

两个字总结：呵呵。

解法

首先判断长度为偶数的情况，先判断两个序列的上中位数大小，如果大小一样，那么直接返回，这两个数必是上中位数。如果不一样，那么判断两个上中位数的大小，如果序列1的较大，那么就递归序列1的前半段和序列2的后半段。直到只有只有两个数的时候截止。两个数找上中位数就返回较小那一个就可以了。

当长度为奇数的时候，同理比较两个数组的上中位数大小，如果一样那么直接返回。如果不一样则比较大小，如果序列1的较大，那么就递归序列1的前半段和序列2的后半段。但是要注意的是，要保持两个的长度相同。其余和偶数情况一样。

代码：

int solve(int a[],int b[],int beginA,int endA,int beginB,int endB){
    int midA = (endA + beginA) / 2;
    int midB = (endB + beginB) / 2;

    cout << a[midA] << " " << b[midB] << endl;

    int offset;
    if((endA - beginA + 1) % 2 == 0)
        offset = 1;
    else 
        offset = 0; 

    if(endA == beginA)
        return a[midA] < b[midB] ? a[midA]: b[midB];

    if(a[midA] > b[midB])
        return solve(a,b,beginA,midA,midB+offset,endB);
    else if(a[midA] < b[midB])
        return solve(a,b,midA+offset,endA,beginB,midB);
    else
        return a[midA];
}

在上述代码中，offset用来控制比较范围的偏差。

第K个数

描述

给定两个有序序列，不一定等长，如何找到这两个序列排序好之后的第K个数。

解法

假设两个序列arr1和arr2的长度分别为m和n，且m>=n。

  1. 当k的值大于m+n或者小于0时，返回异常。
  2. 当k的值小于n时，找到arr1的前k个值和arr2的前k个值的上中位数即可。
  3. 当k的值大于m时，先比较arr1的倒数第k个值和arr2的最后一个值，再比较arr2的倒数第k个值和arr1的最后一个值，具体怎么返回见代码。仍没找到则找arr1的后k-1个值和arr2的后k-1个值的上中位数即可。
  4. 当k的值大于n小于m时，先比较arr1[k-n-1]的值和arr2的最后一个值，如果arr1[k-n-1]大，那么直接返回。如果小的话，找到arr1的k-n到k-1位置和arr2的上中位数即可.

代码：

int solve(int a[],int b[],int n,int m,int k){
    if(n > m)
        return solve(b,a,m,n,k);
    if(k < 0 || k > n + m)
        return -1;
    if(k <= n)
        return findUpMedNumber(a,b,0,k-1,0,k-1);
    if(k > m){
        if(a[k-m-1] >= b[m-1])
            return a[k-n-1];
        else if(b[k-m-1] > a[n-1])
            return b[k-m-1];
        return findUpMedNumber(a,b,k-m,n-1,k-m,m-1);
    }
    if(b[k-n-1] > a[n-1])
        return b[k-n-1];
    return findUpMedNumber(a,b,0,n-1,k-n,k-1);
}

代码中的findUpMedNumber()函数就是上面上中位数的代码。
