左程云代码课记录（二、三）

作者:PJR    发布于: 2018-06-15

看了左神的第二、三堂课，继续记录总结。

生成随机区间函数

描述：给定一个可以生成1-5的随机数的函数，问：如何用这个函数构造一个可以生成1-6随机数的函数。设给定函数为r()。
方法步骤：

  1. r()生成1-5，那么r()-1生成0-4
  2. (r()-1)*5生成0,5,10,15,20
  3. (r()-1)*5 + r()-1生成0-24的随机数
  4. 对3的这个函数做一定处理：当生成的数为21,22,23,24时，就丢弃，重新随机产生数。
  5. 将4的这个函数%6，得到0-5的随机数产生的函数。
  6. 将5的这个函数+1，完成构造。

拓展

描述：给定一个可以生成1-m的随机数的函数，问：如何用这个函数构造一个可以生成1-n随机数的函数。

方法：不断将原函数生成随机数的范围扩大，直到比n大为止。范围总是0->m-1 –>> 0->m^k-1。后面的方法同上。

最短子序列

描述：给定一个无序序列，问：现在要将序列排序成递增序列，那么最短需要排序哪一段子序列之后，序列就排序成了一个递增序列。

方法：首先从最左边开始找，遇到大的值就更新，遇到小的值就记录。从右边来开始找，遇到小的值就更新，遇到大的值就记录。最后最短需要排序的子序列就是两边的最后一次记录之间的部分（包含两个记录的位置）。

划分

描述：给定一个序列，允许你从某一位置将序列划分成两部分，要求两部分的各自的最大值之差最大，求这个差。

方法1：预处理这个序列，得到一个最大序列和最小序列。最大序列表示从左往右每个位置之前（包括当前位置）最大的值，最小序列表示从右往左每个位置之前（包括当前位置）最小的值。然后两个序列对应位置相减，就得到一个差的序列，在差最大位置处划分即可。空间复杂度O(2n),时间复杂度O(3n)
方法2：找到序列的最大值，跟最左端的数做差，再跟最右端的数做差。取较大者。

后继节点

描述：给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

方法步骤：

  1. 二叉树为空，则返回空。
  2. 节点右孩子存在，则设置一个指针从该节点的右孩子出发，一直沿着指向左子结点的指针找到的叶子节点即为下一个节点。
  3. 节点不是根节点。如果该节点是其父节点的左孩子，则返回父节点；否则继续向上遍历其父节点的父节点，重复之前的判断，返回结果。

代码：

TreeLinkNode* GetNext(TreeLinkNode* pNode)
{
    if (pNode == NULL)
        return NULL;

    if (pNode->right != NULL){
        pNode = pNode->right;

        while (pNode->left != NULL)
            pNode = pNode->left;

        return pNode;
    }

    while (pNode->next != NULL){
        TreeLinkNode *proot = pNode->next;

        if (proot->left == pNode)
            return proot;

        pNode = pNode->next;
    }

    return NULL;
}

局部最小

描述：有一个序列A，长度为n,现在给出一个局部最小的定义，当A[0] < A[1]，则A[0]为局部最小。当A[n-1] < A[n-2]，A[n-2]为局部最小。其他位置时，A[i] < A[i-1]且A[i] < A[i+1]时,A[i]为局部最小。现在给定一个序列，找出任意一个局部最小的位置。

方法：二分法。首先看序列左右两边的元素满不满足条件，如果满足，则直接返回，如果不满足，则查看序列中间的节点。如果中间的节点满足条件（比两边的值都小），则返回中间位置，如果不满足，则用二分法查看元素值小于当前位置值的那一边，直到找到局部最小为止。注：如果两边的值都小于中间位置的值，那么二分两边都可以。

代码：

int solve(int a[],int n){
    if(a[0] < a[1])
        return a[0];
    if(a[n-1] < a[n-2])
        return a[n-1];

    int i = 0,j = n-1,mid;
    while(i<j){
        mid = (i+j) / 2;
        if(a[mid] < a[mid+1] && a[mid] > a[mid-1])
            return a[mid];
        if(a[mid] > a[mid-1])
            j = mid;
        if(a[mid] > a[mid+1])
            i = mid;
    }
}

子数组累乘最大乘积

描述：给定数组A，求子数组累乘最大乘积（必须连续）。

方法：典型dp。三种决策:
1.MaxRes[i] = MaxRes[i-1] A[i]
2.MaxRes[i] = MinRes[i-1] A[i]
3.MaxRes[i] = A[i]

代码：

double Max(double a,double b,double c){
    double t = a>b ? a:b;
    return t>c ? t:c;
}

double Min(double a,double b,double c){
    double t = a<b ? a:b;
    return t<c ? t:c;
}

double solve(double num[],int n){
    double max[100],min[100],res;
    max[0] = num[0];
    min[0] = num[0];
    res = num[0];

    double tMax,tMin;
    for(int i=1;i<n;i++){
        tMax = max[i-1] * num[i];
        tMin = min[i-1] * num[i];

        max[i] = Max(tMax,tMin,num[i]);
        min[i] = Min(tMax,tMin,num[i]);

        res = res>max[i] ? res:max[i];
    }
    return res;
}

完全二叉树的节点个数

描述：给定一个完全二叉树的头结点，求完全二叉树的节点个数，要求时间复杂度小于O(N)。

方法步骤：

  1. 首先直接找到左子树的最左节点，用来记录树的高度h。
  2. 查看L层head节点的右子树的最左节点。
  3. 如果这个节点的层数等于h，那么说明左子树全满，左子树的节点数为2^(h-L)-1。再递归结算右子树。
  4. 如果这个节点的层数小于h，那么说明右子树全满(但是只有h-1层)，右子树的高度为2^(h-1-L)。再递归左子树。

代码： 

//判断从head开始树的高度 
int levelOfTree(Node *head){
    if(head == NULL)
        return 0;
    int level = 1;
    while(head->left){
        level++;
        head = head->left;
    }
    return level;
}

//递归计算树的节点个数 
int bs(Node *head,int L,int H){
//  cout << "value: " << head->value << endl;
    if(L == H)
        return 1;
    int h = levelOfTree(head->right);
//  cout << "h: " << h << endl;
//  cout << "L: " << L << endl;
//  cout << "H: " << H << endl;
    if(h+L == H)
        return pow(2,H-L) + bs(head->right,L+1,H);
    else
        return pow(2,H-1-L) + bs(head->left,L+1,H);
}

int solve(Node *head){
    if(head == NULL)
        return 0;

    int H = levelOfTree(head);
    return bs(head,1,H);
}
